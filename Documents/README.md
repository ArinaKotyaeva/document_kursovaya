# Пояснительная записка

Создайте файл Word (.docx) с именем каталога сдачи, например:

`БИВТ-24-1_Иванов_ИВ_9_Документ.docx`

Структура и содержание разделов — в файле `POYASNITELNAYA_ZAPISKA_SODERZHANIE.md` в этой папке.

Оформление по ГОСТ 7.32-2017:
- шрифт Times New Roman 12 pt, код Courier New 12 pt;
- интервал 1,5;
- поля: левое 30 мм, правое 15 мм, верх/низ 20 мм;
- до 25 страниц;
- нумерация страниц, колонтитул с ФИО и темой;
- ссылки на литературу [N] и ссылка на git-репозиторий.
